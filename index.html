Remed7

Readme files for your CTF

Remed7 is a simple Python3 projects which will create a Readme.md file with your TARGET IP address and save it to the directory of your choice. It can also read Nmap outputs and write it directly to the file.

Remed7 meant to make your CTF easier. You can download the first version soon from my repo.
